⚡ 菜单项不显示？立即解决！
================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔥 最快解决方案（90%有效）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ 在Unity中按快捷键：Ctrl + R
   （这会强制刷新资源）

2️⃣ 等待10-20秒，让Unity编译完成
   （看右下角是否有旋转的圆圈）

3️⃣ 编译完成后，查看顶部菜单栏
   应该能看到"TreePlanQAQ"菜单了

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 验证菜单是否正常
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

点击：TreePlanQAQ → Test Menu Item

如果弹出"测试成功"对话框，说明菜单系统正常！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎯 然后使用这些功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TreePlanQAQ菜单下应该有：

✓ Test Menu Item ................... 测试菜单
✓ Generate Plant Models ............ 通用植物生成器
✓ Generate Orange Tree Models ...... 橘子树窗口生成器
✓ Quick Generate All Orange Trees .. 一键生成橘子树 ⭐
✓ Open Orange Trees Folder ......... 打开输出文件夹

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚀 推荐使用方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式A（最快）：
TreePlanQAQ → Quick Generate All Orange Trees

方式B（可视化）：
TreePlanQAQ → Generate Orange Tree Models
然后在窗口中点击"生成所有橘子树模型"

两种方式效果完全相同！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❌ 还是看不到菜单？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开Console窗口（Window → General → Console）
2. 看看是否有红色错误
3. 如果有错误，告诉我错误内容

或者：

1. 关闭Unity
2. 重新打开项目
3. 等待编译完成

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 提示：Unity第一次编译新脚本可能需要一点时间，
请耐心等待右下角的编译图标消失！
