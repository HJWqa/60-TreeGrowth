🍊 橘子树超简单生成方法
========================================

只需3步，不需要任何菜单！

========================================
第1步：创建预览物体
========================================

在Unity中：
1. Hierarchy窗口 → 右键 → Create Empty
2. 命名为 "OrangeTreePreview"
3. 在Inspector中点击 "Add Component"
4. 搜索并添加：OrangeTreePreview
   （注意：是 OrangeTreePreview，不是 OrangeTreeQuickGenerator）

========================================
第2步：生成所有橘子树
========================================

在Inspector中设置：
1. 勾选 "Show All Stages" ✓
2. 右键点击 OrangeTreePreview 组件标题
3. 选择 "生成所有阶段"

✅ 完成！场景中会出现10棵橘子树，横向排列

========================================
第3步：保存为预制体
========================================

1. 右键点击 OrangeTreePreview 组件
2. 选择 "保存为预制体"
3. 等待保存完成

✅ 所有预制体自动保存到：
   Assets/Prefabs/Plants/OrangeTrees/

========================================
📝 右键菜单说明
========================================

在 OrangeTreePreview 组件上右键，会看到：

✅ 生成所有阶段
   - 生成全部10个橘子树阶段
   - 横向排列

✅ 生成当前阶段
   - 只生成 Preview Stage 选择的阶段

✅ 清除所有模型
   - 删除场景中的所有树模型

✅ 保存为预制体
   - 将场景中的树保存为预制体
   - 自动创建文件夹
   - 自动命名

========================================
⚙️ Inspector设置说明
========================================

Preview Stage:
   - 选择要预览的单个阶段
   - 默认：MatureTree（成树）

Generate On Start:
   - 勾选后运行游戏时自动生成
   - 建议：取消勾选（手动生成更好）

Spacing:
   - 树之间的间距
   - 默认：3单位
   - 建议：2-5单位

Show All Stages:
   - 勾选后生成所有10个阶段
   - 不勾选只生成单个阶段

========================================
💡 完整流程
========================================

1. 创建空物体
2. 添加 OrangeTreePreview 组件
3. 勾选 "Show All Stages"
4. 右键组件 → "生成所有阶段"
5. 右键组件 → "保存为预制体"
6. 完成！

========================================
🎯 使用技巧
========================================

技巧1：快速预览
--------------
- 勾选 Show All Stages
- 右键 → 生成所有阶段
- 在Scene视图中查看所有阶段

技巧2：单独查看
--------------
- 取消勾选 Show All Stages
- 选择 Preview Stage
- 右键 → 生成当前阶段

技巧3：调整间距
--------------
- 修改 Spacing 值
- 右键 → 清除所有模型
- 右键 → 生成所有阶段
- 查看新的间距效果

技巧4：重新生成
--------------
如果不满意：
- 右键 → 清除所有模型
- 调整设置
- 右键 → 生成所有阶段

========================================
🐛 常见问题
========================================

Q: 找不到 OrangeTreePreview 组件？
A: 
1. 等待Unity编译完成（右下角进度条）
2. 查看Console是否有编译错误
3. 尝试 Assets → Refresh (Ctrl+R)

Q: 右键没有菜单？
A: 
1. 确保右键点击的是组件标题
2. 不是点击字段或空白处
3. 组件标题是蓝色的那一栏

Q: 生成的树看不见？
A: 
1. 在Scene视图中按F键聚焦
2. 检查物体位置是否在视野内
3. 查看Console日志

Q: 保存预制体失败？
A: 
1. 先生成模型
2. 检查是否有子对象
3. 查看Console错误信息

Q: 模型是粉红色？
A: 
检查项目渲染管线设置（HDRP/URP/Built-in）
