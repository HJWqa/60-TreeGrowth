# Blender橘子单独上色教程

## 前提条件
- 已有完整的橘子树模型（树干、叶子、橘子都在一个对象中）
- 需要给橘子部分单独设置颜色

---

## 🔥 快速方法：已有材质的模型（你的情况）

如果你的模型已经有细分网格和材质（像图片中那样），直接修改材质颜色最简单：

### 步骤1：选择橘子对象
1. 在右上角大纲视图（Outliner）中找到橘子对象
2. 或者直接在3D视图中点击选中一个橘子

### 步骤2：进入材质编辑
1. 切换到 **Shading** 工作区（顶部标签栏）
2. 或按 `Shift + F3` 快速切换

### 步骤3：修改橘子颜色
在下方的 Shader Editor 中：
1. 找到橘子的材质节点（通常叫 "Orange" 或 "Fruit"）
2. 找到 **Principled BSDF** 节点
3. 点击 **Base Color** 旁边的色块
4. 在弹出的拾色器中选择你想要的颜色：
   - 橙色：RGB(1.0, 0.5, 0.0)
   - 黄色：RGB(1.0, 0.9, 0.0)
   - 红色：RGB(1.0, 0.2, 0.0)
   - 绿色（未成熟）：RGB(0.5, 0.8, 0.3)

### 步骤4：实时预览
1. 在3D视图右上角，切换到 **Material Preview** 模式（第三个球体图标）
2. 或按 `Z` 键，选择 "Material Preview"
3. 立即看到颜色变化效果

### 步骤5：批量修改所有橘子
如果所有橘子共用一个材质：
- 修改一次，所有橘子都会变色

如果橘子是独立对象：
1. 选中所有橘子对象（按住 `Shift` 点击）
2. 最后选中已修改颜色的橘子（作为活动对象）
3. 按 `Ctrl + L` → 选择 "Link Materials"（链接材质）
4. 所有橘子使用相同材质

## 方法一：通过面选择分离橘子（推荐）

### 步骤1：进入编辑模式
1. 选中橘子树模型
2. 按 `Tab` 键进入编辑模式
3. 按 `Alt + A` 取消全选

### 步骤2：选择橘子部分
1. 按 `3` 键切换到面选择模式
2. 按住 `Alt` 键，点击橘子的一个面（会选中整个橘子球体）
3. 按住 `Shift` 键，继续 `Alt + 点击` 其他橘子，选中所有橘子

**技巧**：如果橘子是独立的球体，可以：
- 按 `L` 键（鼠标悬停在橘子上）选择连接的面
- 或使用 `Shift + L` 选择多个独立部分

### 步骤3：分离橘子对象
1. 选中所有橘子面后，按 `P` 键
2. 选择 "Selection"（按选择分离）
3. 现在橘子变成了独立对象

### 步骤4：为橘子创建材质
1. 按 `Tab` 退出编辑模式
2. 选中分离出来的橘子对象
3. 切换到 Shading 工作区（顶部标签）
4. 在右侧属性面板找到材质属性（球体图标）
5. 点击 "+" 添加新材质，命名为 "Orange_Material"

### 步骤5：设置橘子颜色
在 Shader Editor 中：
1. 找到 "Principled BSDF" 节点
2. 设置 Base Color（基础色）：
   - R: 1.0, G: 0.5, B: 0.0（橙色）
   - 或使用拾色器选择你喜欢的橙色
3. 调整其他参数：
   - Roughness（粗糙度）: 0.3-0.5（橘子表面有点粗糙）
   - Subsurface（次表面散射）: 0.1-0.2（让橘子有半透明感）
   - Subsurface Color: 浅橙色

## 方法二：使用顶点组和材质槽

### 步骤1：创建顶点组
1. 选中橘子树，按 `Tab` 进入编辑模式
2. 选择所有橘子的面（参考方法一步骤2）
3. 在右侧属性面板找到 "Object Data Properties"（绿色三角形图标）
4. 展开 "Vertex Groups"，点击 "+" 添加新组
5. 命名为 "Oranges"，点击 "Assign"

### 步骤2：添加材质槽
1. 按 `Tab` 退出编辑模式
2. 在材质属性面板：
   - 点击 "+" 添加新材质槽
   - 点击 "New" 创建新材质，命名 "Orange_Material"
   - 设置橙色（参考方法一步骤5）

### 步骤3：分配材质到橘子
1. 按 `Tab` 进入编辑模式
2. 选中所有橘子面
3. 在材质槽列表中选择 "Orange_Material"
4. 点击 "Assign" 按钮

## 推荐的橘子材质参数

```
Base Color: RGB(1.0, 0.5, 0.0) 或 Hex #FF8000
Metallic: 0.0
Roughness: 0.4
Subsurface: 0.15
Subsurface Radius: RGB(1.0, 0.5, 0.2)
Subsurface Color: RGB(1.0, 0.6, 0.2)
```

## 进阶：添加橘子纹理

如果想要更真实的效果：

1. 添加 Image Texture 节点（Shift + A → Texture → Image Texture）
2. 加载橘子皮纹理图片
3. 连接到 Base Color 或使用 Mix RGB 节点混合
4. 添加 Bump 节点制作凹凸效果

## 常见问题

### Q: 选择橘子时选中了叶子怎么办？
A: 按住 `Shift` 点击不需要的面取消选择，或使用 `Alt + A` 取消全选重新选择

### Q: 分离后橘子位置变了？
A: 分离不会改变位置，如果发生了，按 `Ctrl + Z` 撤销重新操作

### Q: 想要不同颜色的橘子（成熟度不同）？
A: 可以分别选择不同的橘子，分配不同的材质槽，设置不同的颜色：
- 青色（未成熟）: RGB(0.5, 0.8, 0.3)
- 橙色（成熟）: RGB(1.0, 0.5, 0.0)
- 深橙（过熟）: RGB(0.9, 0.3, 0.0)

## 导出到Unity注意事项

1. 如果分离了橘子对象，导出FBX时确保选中所有对象
2. 在Unity中需要重新设置材质
3. 建议在Blender中保持对象层级清晰：
   - OrangeTree（父对象）
     - Trunk（树干）
     - Leaves（叶子）
     - Oranges（橘子）

## 快捷键总结

- `Tab`: 切换编辑/对象模式
- `3`: 面选择模式
- `Alt + 点击`: 选择循环边/连接面
- `L`: 选择连接的几何体
- `P`: 分离选择
- `Shift + A`: 添加节点
- `Ctrl + Z`: 撤销
