# 左上角图标设置说明

## 功能概述

在游戏左上角创建三个功能图标按钮：

1. **⚙ 齿轮图标** - 游戏基础设置
2. **! 感叹号图标** - 温度湿度提示
3. **▶ 三角形图标** - 游戏暂停/继续

## 快速设置

### 自动创建（推荐）

1. 在 Unity 菜单栏选择：`橘子树 > UI设置 > 创建左上角图标`
2. 点击"创建左上角图标"按钮
3. 完成！所有图标和面板自动创建并配置

### 手动创建

如果需要手动创建，请参考下面的详细步骤。

## 图标功能详解

### 1. ⚙ 设置按钮

**外观：** 灰色圆圈 + 白色齿轮图标

**功能：** 打开/关闭游戏设置面板

**面板内容：**
- 音量设置
- 画质设置
- 其他游戏选项

**交互：**
- 点击打开设置面板
- 再次点击关闭面板
- 打开设置时自动关闭提示面板

### 2. ! 提示按钮

**外观：** 灰色圆圈 + 白色感叹号图标

**功能：** 显示温度湿度的合理范围提示

**面板内容：**
```
温度湿度合理的范围是：

🌡️ 温度：15°C - 30°C
   最佳温度：20°C - 25°C

💧 湿度：40% - 80%
   最佳湿度：50% - 70%

☀️ 光照：400 - 800 lux
   最佳光照：600 lux
```

**交互：**
- 点击打开提示面板
- 再次点击关闭面板
- 打开提示时自动关闭设置面板

### 3. ▶ 暂停/继续按钮

**外观：** 灰色圆圈 + 白色图标

**功能：** 控制游戏的暂停和继续

**状态指示：**
- **▶ 播放图标** - 游戏已暂停（点击可继续）
- **❚❚ 暂停图标** - 游戏正在运行（点击可暂停）

**交互：**
- 点击切换暂停/继续状态
- 暂停时显示播放图标 ▶
- 运行时显示暂停图标 ❚❚

## UI 层级结构

```
Canvas
├── TopIconsContainer（左上角图标容器）
│   ├── SettingsButton（⚙ 设置）
│   │   └── Icon (Text)
│   ├── HintButton（! 提示）
│   │   └── Icon (Text)
│   └── PauseButton（▶ 暂停/继续）
│       └── Icon (Text)
├── SettingsPanel（设置面板，初始隐藏）
│   ├── Title
│   └── Content
└── EnvironmentHintPanel（提示面板，初始隐藏）
    ├── Title
    └── Content
```

## 布局说明

### 图标容器
- **位置：** 左上角
- **锚点：** 左上 (0, 1)
- **偏移：** (20, -20)
- **布局：** 水平排列，间距 15px
- **图标大小：** 50x50 像素
- **圆圈颜色：** 灰色 (0.5, 0.5, 0.5)
- **图标颜色：** 全部白色

### 设置面板
- **位置：** 屏幕正中间
- **锚点：** 中心 (0.5, 0.5)
- **大小：** 400x300
- **背景色：** 深灰色半透明
- **初始状态：** 隐藏

### 提示面板
- **位置：** 屏幕正中间
- **锚点：** 中心 (0.5, 0.5)
- **大小：** 500x350
- **背景色：** 深绿色半透明
- **初始状态：** 隐藏

## 脚本说明

### TopIconsController.cs

**主要功能：**
- 管理三个图标按钮的点击事件
- 控制面板的显示/隐藏
- 更新暂停按钮的状态

**公共方法：**
- `CloseAllPanels()` - 关闭所有面板

**Inspector 字段：**
- Settings Button - 设置按钮引用
- Hint Button - 提示按钮引用
- Pause Button - 暂停按钮引用
- Pause Icon Text - 暂停按钮图标文本引用
- Settings Panel - 设置面板引用
- Hint Panel - 提示面板引用
- Tree Controller - 树控制器引用

## 工作流程

### 设置按钮流程
```
点击设置按钮
  ↓
切换设置面板显示状态
  ↓
如果打开设置 → 关闭提示面板
```

### 提示按钮流程
```
点击提示按钮
  ↓
切换提示面板显示状态
  ↓
如果打开提示 → 关闭设置面板
```

### 暂停按钮流程
```
点击暂停按钮
  ↓
切换树控制器的暂停状态
  ↓
更新图标符号
  ↓
❚❚ 暂停图标（运行中）⟷ ▶ 播放图标（已暂停）
```

## 自定义修改

### 修改图标样式

在 `SetupTopIcons.cs` 的 `CreateIconButton` 方法中修改：

```csharp
// 修改图标大小
rect.sizeDelta = new Vector2(60, 60); // 改为 60x60

// 修改圆圈颜色
Color circleColor = new Color(0.3f, 0.3f, 0.3f); // 深灰色

// 修改图标颜色
Color iconColor = new Color(1f, 0.5f, 0.2f); // 橙色

// 修改图标符号
text.text = "★"; // 改为星星
```

### 修改面板内容

在 `SetupTopIcons.cs` 中修改对应的 `CreateSettingsPanel` 或 `CreateHintPanel` 方法。

### 添加更多功能

在 `TopIconsController.cs` 中添加新的按钮和方法：

```csharp
[SerializeField] private Button newButton;

private void OnNewButtonClicked()
{
    // 你的功能代码
}
```

## 常见问题

**Q: 图标显示不正确？**
A: 检查字体是否支持 Unicode 符号，可以尝试使用其他字体

**Q: 按钮点击没反应？**
A: 检查 Inspector 中的引用是否正确设置

**Q: 面板位置不对？**
A: 调整面板的 RectTransform 的 Anchored Position

**Q: 暂停按钮状态不对？**
A: 检查 Pause Icon Text 字段是否正确引用了按钮的 Icon 文本组件

**Q: 图标符号不显示？**
A: 某些 Unicode 符号可能不被字体支持，尝试使用其他符号或字体

**Q: 想要修改提示内容？**
A: 在 SetupTopIcons.cs 的 CreateHintPanel 方法中修改 contentText.text
