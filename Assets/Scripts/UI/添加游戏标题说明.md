# 添加"种植果树大挑战"标题

## 方法1：手动创建（推荐，2分钟）

### 步骤：

1. **在Hierarchy中右键Canvas**
   - UI → Text

2. **重命名为 "GameTitle"**

3. **设置RectTransform（位置）**
   - Anchor Presets: 点击左上角的方框
   - 按住 Alt + Shift，点击顶部中心位置
   - 或手动设置：
     - Anchor Min: (0.5, 1)
     - Anchor Max: (0.5, 1)
     - Pivot: (0.5, 1)
   - Pos X: 0
   - Pos Y: -50（距离顶部50像素）
   - Width: 800
   - Height: 100

4. **设置Text组件（文字）**
   - Text: `种植果树大挑战`
   - Font Size: 60
   - Alignment: Center & Middle（水平和垂直居中）
   - Color: 白色（或你喜欢的颜色）
   - Font Style: Bold（加粗）

5. **添加阴影效果（可选）**
   - 选中GameTitle
   - Add Component → Shadow
   - Effect Color: 黑色半透明 (0, 0, 0, 128)
   - Effect Distance: (3, -3)

6. **添加描边效果（可选）**
   - Add Component → Outline
   - Effect Color: 黑色 (0, 0, 0, 200)
   - Effect Distance: (2, -2)

7. **完成！**

## 方法2：使用脚本自动创建

### 步骤：

1. **选中Canvas**
2. **Add Component → Game Title UI**
3. **设置参数**（可选，使用默认值即可）：
   - Title Text: 种植果树大挑战
   - Font Size: 60
   - Text Color: 白色
   - Bold: ✓
   - Add Shadow: ✓
   - Add Outline: ✓
   - Top Offset: 50
4. **运行游戏**，标题会自动创建

## 样式建议

### 颜色方案1：经典白色
- Text Color: 白色 (255, 255, 255)
- Shadow: 黑色半透明
- Outline: 黑色

### 颜色方案2：金色
- Text Color: 金色 (255, 215, 0)
- Shadow: 深棕色
- Outline: 深金色

### 颜色方案3：绿色（自然主题）
- Text Color: 亮绿色 (144, 238, 144)
- Shadow: 深绿色
- Outline: 深绿色

### 颜色方案4：渐变效果
- 使用两个Text叠加
- 底层：深色描边
- 顶层：亮色文字

## 字体大小建议

根据屏幕分辨率：
- 1920x1080: Font Size 60
- 1280x720: Font Size 40
- 2560x1440: Font Size 80

或使用Canvas Scaler自动缩放。

## 进阶美化

### 1. 添加背景装饰
```
GameTitle
├── Text (标题文字)
└── Background (装饰背景)
    └── Image (半透明黑色背景)
```

创建步骤：
1. 右键GameTitle → UI → Image
2. 重命名为 Background
3. 设置为GameTitle的第一个子对象（拖到Text上方）
4. 设置：
   - Color: 黑色半透明 (0, 0, 0, 100)
   - 大小稍大于文字

### 2. 添加动画效果

创建简单的缩放动画：
1. 选中GameTitle
2. Window → Animation → Animation
3. 创建新动画 "TitlePulse"
4. 添加关键帧：
   - 0.0s: Scale (1, 1, 1)
   - 0.5s: Scale (1.05, 1.05, 1)
   - 1.0s: Scale (1, 1, 1)
5. 设置循环播放

### 3. 添加图标

在标题旁边添加树的图标：
1. 准备一个树的图标（PNG，透明背景）
2. 导入到Unity
3. 右键GameTitle → UI → Image
4. 重命名为 TreeIcon
5. 设置：
   - Source Image: 拖入树图标
   - Pos X: -250（在标题左侧）
   - Size: 80x80

### 4. 使用TextMeshPro（更好的文字效果）

如果想要更好的文字渲染：
1. 右键Canvas → UI → Text - TextMeshPro
2. 如果提示导入TMP Essentials，点击Import
3. 设置同上，但效果更清晰

## 响应式设计

确保标题在不同分辨率下都显示正常：

1. **设置Canvas Scaler**
   - 选中Canvas
   - Canvas Scaler组件
   - UI Scale Mode: Scale With Screen Size
   - Reference Resolution: 1920 x 1080
   - Screen Match Mode: Match Width Or Height
   - Match: 0.5

2. **使用Anchor**
   - 标题使用顶部中心锚点
   - 这样无论屏幕大小，标题都在顶部中间

## 常见问题

### Q: 标题显示不全？
A: 增加Width值，或减小Font Size

### Q: 标题位置不对？
A: 检查Anchor设置，应该是顶部中心

### Q: 文字模糊？
A: 
1. 使用TextMeshPro代替Text
2. 增加Canvas的Render Mode质量
3. 确保Font Size不要太小

### Q: 想要标题居左或居右？
A: 修改Anchor：
- 居左：Anchor Min/Max (0, 1)
- 居右：Anchor Min/Max (1, 1)
