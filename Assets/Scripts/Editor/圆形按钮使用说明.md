# 圆形按钮+三角形图标 使用说明

## 效果预览

```
温度控制：
  ⏶  ← 圆形背景 + 正三角形图标
25.0°C
  ⏷  ← 圆形背景 + 倒三角形图标

湿度控制：
  ⏶  ← 圆形背景 + 正三角形图标
60.0%
  ⏷  ← 圆形背景 + 倒三角形图标
```

## 一键自动转换（30秒完成）

### 步骤：

1. **打开Unity编辑器**

2. **打开工具窗口**
   - 菜单：`TreePlanQAQ > UI Tools > Convert Buttons to Shapes`

3. **点击按钮**
   - 点击 `🎯 自动设置所有UI形状（推荐）`

4. **完成！**
   - 所有按钮自动转换为圆形背景
   - 温度/湿度按钮自动添加三角形图标

## 按钮命名要求

工具会根据按钮名称自动识别类型：

### 温度增加按钮
命名包含：`TempUp`, `温度上`, `temperature_up`, `temp_increase`
→ 转换为：🔴 红色圆形背景 + ⏶ 白色正三角形

### 温度减少按钮
命名包含：`TempDown`, `温度下`, `temperature_down`, `temp_decrease`
→ 转换为：🔵 蓝色圆形背景 + ⏷ 白色倒三角形

### 湿度增加按钮
命名包含：`HumidityUp`, `湿度上`, `humidity_up`, `humid_increase`
→ 转换为：🔴 红色圆形背景 + ⏶ 白色正三角形

### 湿度减少按钮
命名包含：`HumidityDown`, `湿度下`, `humidity_down`, `humid_decrease`
→ 转换为：🔵 蓝色圆形背景 + ⏷ 白色倒三角形

### 暂停/重置按钮
命名包含：`Pause`, `Reset`, `暂停`, `重置`
→ 转换为：🟢 绿色/🟠 橙色纯圆形按钮（带文字）

## 转换结果

### 圆形+三角形按钮结构
```
Button (圆形背景)
├── Image组件 (圆形Sprite)
└── TriangleIcon (子对象)
    └── TriangleImage组件 (三角形)
```

### 默认设置
- **按钮大小**：60x60 像素（圆形）
- **三角形大小**：30x30 像素
- **正三角按钮颜色**：浅红色 (255, 128, 128)
- **倒三角按钮颜色**：浅蓝色 (128, 179, 255)
- **三角形图标颜色**：白色

## 手动调整

转换后可以在Inspector中调整：

### 调整圆形背景颜色
1. 选中按钮
2. 找到 Image 组件
3. 修改 Color 属性

### 调整三角形颜色
1. 选中按钮
2. 展开子对象 TriangleIcon
3. 找到 TriangleImage 组件
4. 修改 Triangle Color 属性

### 调整三角形大小
1. 选中 TriangleIcon 子对象
2. 修改 RectTransform 的 Width 和 Height

### 调整按钮大小
1. 选中按钮
2. 修改 RectTransform 的 Width 和 Height
3. 建议保持正方形（Width = Height）以保持圆形

## 常见问题

### Q: 转换后三角形不显示？
A: 检查TriangleImage脚本是否编译成功。打开Console查看是否有编译错误。

### Q: 按钮点击没反应？
A: 检查Button组件的Target Graphic是否设置为Image组件（圆形背景）。

### Q: 想要更大的三角形？
A: 选中TriangleIcon子对象，增加RectTransform的Size。建议范围：20-40像素。

### Q: 想要不同的颜色？
A: 在Inspector中修改Image组件（背景）和TriangleImage组件（图标）的Color属性。

### Q: 可以改变三角形方向吗？
A: 选中TriangleIcon，在TriangleImage组件中勾选/取消勾选 Point Up。

## 推荐布局

### 垂直布局（温度/湿度组）
```
[  ⏶  ]  ← 增加按钮（圆形+正三角）
25.0°C   ← 数值显示
[  ⏷  ]  ← 减少按钮（圆形+倒三角）
```

### 设置步骤
1. 创建空对象作为组容器
2. 添加三个子对象：
   - UpButton（增加按钮）
   - ValueText（数值文本）
   - DownButton（减少按钮）
3. 使用工具转换按钮
4. 调整位置和间距

## 完整示例

假设你要创建温度控制UI：

1. **创建结构**
   ```
   Canvas
   └── TempGroup
       ├── TempUpButton
       ├── TempText
       └── TempDownButton
   ```

2. **运行转换工具**
   - `TreePlanQAQ > UI Tools > Convert Buttons to Shapes`
   - 点击 `自动设置所有UI形状`

3. **结果**
   - TempUpButton → 🔴 圆形 + ⏶
   - TempDownButton → 🔵 圆形 + ⏷

4. **调整位置**
   - TempUpButton: Pos Y = 40
   - TempText: Pos Y = 0
   - TempDownButton: Pos Y = -40

5. **完成！**

## 颜色建议

### 温度按钮
- 增加：红色系 (255, 100, 100)
- 减少：蓝色系 (100, 150, 255)

### 湿度按钮
- 增加：蓝色系 (100, 180, 255)
- 减少：深蓝系 (80, 120, 200)

### 三角形图标
- 统一使用白色 (255, 255, 255)
- 或使用对比色以提高可见度

## 进阶技巧

### 添加悬停效果
1. 选中按钮
2. 在Button组件中设置：
   - Highlighted Color: 稍亮的颜色
   - Pressed Color: 稍暗的颜色

### 添加阴影
1. 选中按钮
2. Add Component → Shadow
3. 设置 Effect Distance: (2, -2)

### 添加动画
1. 选中按钮
2. Add Component → Animator
3. 创建按下缩放动画

## 技术说明

### 圆形背景实现
使用Unity内置的 `UI/Skin/Knob.psd` sprite，这是一个完美的圆形图片。

### 三角形图标实现
使用自定义的 `TriangleImage` 组件，通过代码生成三角形网格。

### 层级结构
- 按钮本身：圆形背景（可点击）
- 子对象：三角形图标（仅显示）
- Target Graphic：指向圆形背景

这样设计的好处：
- 点击区域是整个圆形
- 三角形只是视觉装饰
- 可以独立调整背景和图标
