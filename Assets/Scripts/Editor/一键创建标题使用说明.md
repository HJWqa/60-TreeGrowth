# 一键创建游戏标题使用说明

## 🎯 最快方法（10秒完成）

### 在Unity中操作：

1. **打开菜单**：`TreePlanQAQ > UI Tools > Quick Create Title (Default Settings)`
2. **完成！**

就这么简单！标题"种植果树大挑战"会自动出现在Canvas顶部。

## 🎨 自定义标题（30秒）

如果你想自定义标题样式：

1. **打开菜单**：`TreePlanQAQ > UI Tools > Create Game Title`
2. **在弹出窗口中设置**：
   - 标题文字：输入你想要的文字
   - 字体大小：拖动滑块（20-120）
   - 文字颜色：点击选择颜色
   - 距离顶部：调整位置（0-200）
   - 添加阴影：勾选/取消
   - 添加描边：勾选/取消
3. **点击"创建标题"按钮**
4. **完成！**

## 📋 功能特性

### 自动处理
- ✅ 自动查找Canvas（如果没有会自动创建）
- ✅ 自动创建EventSystem（如果需要）
- ✅ 自动设置正确的位置（顶部中心）
- ✅ 自动添加阴影和描边效果
- ✅ 检测重复标题（避免创建多个）

### 默认设置
- 标题文字：种植果树大挑战
- 字体大小：60
- 文字颜色：白色
- 位置：距离顶部50像素
- 阴影：✓ 黑色半透明
- 描边：✓ 黑色

## 🔧 创建后的调整

标题创建后，你可以在Inspector中继续调整：

### 修改文字
1. 选中 GameTitle
2. Text组件 → Text字段
3. 输入新文字

### 修改颜色
1. 选中 GameTitle
2. Text组件 → Color
3. 选择新颜色

### 修改大小
1. 选中 GameTitle
2. Text组件 → Font Size
3. 输入新数值

### 修改位置
1. 选中 GameTitle
2. RectTransform → Pos Y
3. 调整数值（负数向下移动）

### 添加/删除效果
- 添加阴影：Add Component → Shadow
- 添加描边：Add Component → Outline
- 删除效果：右键组件 → Remove Component

## ⚠️ 常见问题

### Q: 点击菜单没反应？
A: 
1. 检查脚本是否编译成功（查看Console）
2. 确保脚本在 Assets/Scripts/Editor 文件夹中
3. 重启Unity编辑器

### Q: 标题位置不对？
A: 
1. 选中GameTitle
2. 在Inspector中调整 Pos Y 值
3. 或重新运行创建工具，调整"距离顶部"参数

### Q: 文字显示不全？
A: 
1. 选中GameTitle
2. RectTransform → Width 增加数值
3. 或减小字体大小

### Q: 想要创建多个标题？
A: 
1. 第一次创建后，重命名为其他名字
2. 再次运行创建工具
3. 或手动复制GameTitle并修改

### Q: 标题被其他UI遮挡？
A: 
1. 在Hierarchy中调整顺序
2. 将GameTitle拖到最下方（最后渲染）
3. 或调整Canvas的Sort Order

## 🚀 进阶技巧

### 1. 添加动画

创建缩放动画：
1. 选中GameTitle
2. Window → Animation → Animation
3. 创建新动画
4. 添加Scale关键帧：
   - 0.0s: (1, 1, 1)
   - 0.5s: (1.05, 1.05, 1)
   - 1.0s: (1, 1, 1)
5. 设置循环

### 2. 添加背景

给标题添加半透明背景：
1. 右键GameTitle → UI → Image
2. 重命名为Background
3. 拖到Text上方（在Hierarchy中）
4. 设置：
   - Color: 黑色半透明 (0, 0, 0, 100)
   - 大小比文字稍大

### 3. 添加图标

在标题旁边添加图标：
1. 准备图标PNG文件
2. 导入到Unity
3. 右键GameTitle → UI → Image
4. 重命名为Icon
5. 设置：
   - Source Image: 拖入图标
   - Pos X: -300（在标题左侧）
   - Size: 80x80

### 4. 使用更好的字体

使用自定义字体：
1. 导入TTF字体文件到Unity
2. 选中GameTitle
3. Text组件 → Font
4. 拖入自定义字体

### 5. 响应式设计

确保在不同分辨率下都正常：
1. 选中Canvas
2. Canvas Scaler组件
3. UI Scale Mode: Scale With Screen Size
4. Reference Resolution: 1920 x 1080
