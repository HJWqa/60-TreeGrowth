## 下一步操作

### 1. 清理场景中的旧对象

在Unity中：
1. 菜单: `TreePlanQAQ/Cleanup Old System`
2. 点击"扫描旧对象"查看需要删除的对象
3. 点击"删除所有旧对象"清理场景
4. 保存场景

### 2. 使用新系统

1. 菜单: `TreePlanQAQ/Orange Tree Setup Wizard`
2. 按照向导步骤配置系统
3. 创建新的橘子树系统
4. 测试运行
