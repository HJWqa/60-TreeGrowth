# 橘子树系统 - 快速开始指南 🚀

## 3步开始使用

### 第1步：清理旧系统 🧹
```
Unity 菜单: TreePlanQAQ > Cleanup Old System
点击: "删除所有旧对象"
保存场景: Ctrl+S
```

### 第2步：创建新系统 🌱
```
Unity 菜单: TreePlanQAQ > Orange Tree Setup Wizard
1. 点击 "创建新配置"
2. 拖拽8个模型到对应槽位
3. 点击 "创建橘子树系统"
```

### 第3步：测试运行 ▶️
```
点击 Play 按钮
观察橘子树生长
```

## 常用菜单

| 菜单项 | 功能 |
|--------|------|
| TreePlanQAQ > Orange Tree Setup Wizard | 创建新系统 |
| TreePlanQAQ > Cleanup Old System | 清理旧系统 |

## 调整参数

### 调整生长速度
```
选中: Assets/Configs/OrangeTreeConfig
修改: Base Growth Rate
- 增大 = 生长更快
- 减小 = 生长更慢
```

### 调整环境影响
```
选中: Assets/Configs/OrangeTreeConfig
修改权重:
- Temperature Weight (温度影响)
- Humidity Weight (湿度影响)
- Sunlight Weight (光照影响)
```

### 手动控制生长
```
选中: Hierarchy > OrangeTree
在 Inspector 中拖动: Current Growth 滑块
```

## 故障排除

### 问题：树不生长
**解决：**
1. 检查 Config 是否已配置
2. 检查 Base Growth Rate > 0
3. 检查环境参数是否合理

### 问题：模型不显示
**解决：**
1. 检查模型是否已导入
2. 检查 Stages 列表中的引用
3. 使用 Check Scene Status 检查

### 问题：菜单项不显示
**解决：**
1. 检查 Console 是否有编译错误
2. 重启 Unity
3. 确认脚本在 Editor 目录下
