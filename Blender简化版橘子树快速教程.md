# Blender 简化版橘子树快速教程

> 性能友好，30分钟完成，适合低配电脑

## 核心理念

- **少量物体**：树干+3-5个树枝+树冠球体
- **不用粒子系统**：手动放置少量叶子
- **低多边形**：保持模型简单
- **先保存**：每5分钟保存一次！

---

## 第一步：树干（5分钟）

### 创建树干
1. **Shift + A** → Mesh → Cylinder
2. 设置：
   - Vertices: **12**（够用就行）
   - Radius: 0.2
   - Depth: 2.5
3. **立即保存**：Ctrl + S → 命名 `SimpleOrangeTree.blend`

### 简单变形
1. **Tab** 进入编辑模式
2. **3** 切换到面选择
3. 选中顶部面 → **S** → **0.7**（收窄）
4. 选中底部面 → **S** → **1.2**（加粗）
5. **Tab** 退出编辑模式

### 添加树皮纹理（可选）
1. 右侧 → Modifier Properties（扳手图标）
2. Add Modifier → Subdivision Surface
   - Levels: **1**（不要太高！）
3. **保存**：Ctrl + S

---

## 第二步：树枝（10分钟）

### 方法：直接从树干挤出

1. **选中树干**
2. **Tab** 进入编辑模式
3. **3** 切换到面选择模式

### 创建第一根树枝
1. 在树干上半部分选一个面
2. **I** 键（Inset）内插 → 缩小面到 0.3 左右
3. **E** 键（Extrude）挤出
4. **S** 键缩放到 0.8
5. **R** 键旋转，调整角度（向上倾斜45度）
6. 重复 **E** → **S** 2-3次，创建分段

### 创建3-4根树枝
1. 在树干不同位置重复上述步骤
2. 树枝分布：
   - 2根向左右
   - 1-2根向前后
3. **保存**：Ctrl + S

### 快速平滑
1. **Tab** 退出编辑模式
2. 右键 → **Shade Smooth**
3. **保存**

---

## 第三步：树冠（5分钟）

### 用一个大球体代替所有叶子

1. **Shift + A** → Mesh → UV Sphere
2. 设置：
   - Segments: **16**（低多边形）
   - Rings: **8**
3. **S** 缩放到 1.5
4. **G** → **Z** 移动到树枝顶部
5. **S** → **Z** → **0.8**（压扁一点）

### 变形成不规则树冠
1. **Tab** 进入编辑模式
2. **O** 键启用比例编辑
3. **1** 切换到顶点模式
4. 随机选择几个顶点
5. **G** 移动，制造不规则感
6. **Tab** 退出

### 设置绿色材质
1. 右侧 → Material Properties（球体图标）
2. **New** 创建材质
3. Base Color: **深绿色** RGB(0.2, 0.5, 0.2)
4. Roughness: **0.7**
5. **保存**

---

## 第四步：橘子（5分钟）

### 创建一个橘子
1. **Shift + A** → Mesh → UV Sphere
2. 设置：
   - Segments: **16**
   - Rings: **8**
3. **S** 缩放到 0.12
4. **S** → **Z** → **0.9**（稍微压扁）

### 橙色材质
1. Material Properties → **New**
2. Base Color: **橙色** RGB(1.0, 0.5, 0.1)
3. Roughness: **0.4**
4. Specular: **0.5**

### 手动复制橘子（只放5-8个）
1. **Shift + D** 复制
2. **G** 移动到树枝末端
3. 重复5-8次
4. 分布在不同树枝上
5. **保存**

---

## 第五步：可选叶子（5分钟）

### 手动放置少量叶子（10-15片就够）

1. **Shift + A** → Mesh → Plane
2. **S** 缩放到 0.15
3. **S** → **Y** → **1.5**（椭圆形）
4. **R** → **X** → **90**（竖起来）

### 叶子材质
1. Material Properties → **New**
2. Base Color: **深绿色** RGB(0.15, 0.5, 0.15)
3. Roughness: **0.6**

### 复制叶子
1. **Shift + D** 复制
2. **R** 随机旋转
3. **G** 移动到树枝上
4. 只放 **10-15片**（不要太多！）
5. **保存**

---

## 第六步：导出到Unity（5分钟）

### 准备导出
1. **A** 全选所有物体
2. **Ctrl + J** 合并成一个物体（重要！）
3. **Ctrl + A** → **All Transforms**（应用所有变换）
4. **保存**

### 导出FBX
1. **File** → **Export** → **FBX (.fbx)**
2. 设置：
   - ☑ **Selected Objects**
   - Scale: **1.00**
   - Forward: **-Z Forward**
   - Up: **Y Up**
   - ☑ **Apply Modifiers**
3. 保存到：`Unity项目/Assets/Models/SimpleOrangeTree.fbx`

---

## Unity导入设置

### 导入配置
1. 选中FBX文件
2. Inspector → Model：
   - Scale Factor: **1**
   - ☑ **Generate Colliders**（如果需要）
3. Materials：
   - **Extract Materials**
   - 保存到 `Assets/Materials/OrangeTree/`

### 设置HDRP材质
1. 树干材质：
   - Base Color: 棕色 RGB(0.4, 0.25, 0.15)
   - Roughness: 0.8
2. 树冠材质：
   - Base Color: 深绿色 RGB(0.2, 0.5, 0.2)
   - Roughness: 0.7
3. 橘子材质：
   - Base Color: 橙色 RGB(1.0, 0.5, 0.1)
   - Roughness: 0.4

## 常见问题

**Q: Blender卡顿怎么办？**
A: 
- 切换到Solid着色模式（Z键 → Solid）
- 减少物体数量
- 关闭修改器预览
- 降低Subdivision Surface级别

**Q: 导出后Unity中模型太大/太小？**
A: 
- Blender中：Ctrl + A → Scale（应用缩放）
- Unity中：调整Import Settings的Scale Factor

**Q: 材质是粉红色？**
A: 
- 需要手动设置HDRP材质
- Extract Materials后重新配置

**Q: 想要更多细节但不卡？**
A: 
- 使用法线贴图（Normal Map）而不是真实几何体
- 使用纹理而不是粒子系统

## 附加：橘子花建模（开花阶段）

### 橘子花特征
- 白色五瓣花
- 花瓣圆润
- 中心有黄色花蕊
- 直径约2-3厘米

---

## 方法1：超简化版（推荐）

### 用一个白色小球代替

最简单的方法，性能友好：

1. **Shift + A** → Mesh → UV Sphere
2. 设置：
   - Segments: **8**
   - Rings: **4**
3. **S** 缩放到 **0.03**（很小）
4. **S** → **Z** → **0.5**（压扁）

### 白色材质
1. Material Properties → **New**
2. 命名：**OrangeFlower**
3. 设置：
   ```
   Base Color: 白色 RGB(1.0, 1.0, 1.0)
   Roughness: 0.3
   Specular: 0.5
   Emission: RGB(1.0, 1.0, 0.9) - 淡黄色
   Emission Strength: 0.1（微微发光）
   ```

### 放置
- 手动复制5-10朵
- 放在树枝末端
- 用于"开花阶段"的树模型

---

## 方法2：简单五瓣花（稍微精细）

### 步骤1：创建花瓣

1. **Shift + A** → Mesh → Plane
2. **S** 缩放到 **0.015**
3. **Tab** 进入编辑模式
4. **S** → **Y** → **1.3**（椭圆形花瓣）

### 步骤2：圆润花瓣边缘

1. **3** 切换到面选择
2. 全选（**A**）
3. **I** 键（Inset）内插 → **0.2**
4. **E** 键挤出 → 向上移动一点点
5. **Tab** 退出编辑模式
6. 右键 → **Shade Smooth**

### 步骤3：复制5片花瓣

1. **Shift + D** 复制花瓣
2. **R** → **Z** → **72** → **Enter**（旋转72度）
3. 重复4次，创建5片花瓣
4. 调整位置，让花瓣稍微重叠

### 步骤4：添加花蕊

1. **Shift + A** → Mesh → UV Sphere
2. **S** 缩放到 **0.005**（很小）
3. **G** → **Z** 移动到花瓣中心上方
4. Material Properties → **New**
5. Base Color: **黄色** RGB(1.0, 0.8, 0.0)

### 步骤5：合并花朵

1. **框选所有花瓣和花蕊**（B键框选）
2. **Ctrl + J** 合并成一个物体
3. 命名：**OrangeFlower**

### 步骤6：复制到树上

1. **Shift + D** 复制
2. 放置5-10朵在树枝上
3. **R** 随机旋转
4. **S** 随机缩放（0.8-1.2倍）

---

## 方法3：使用纹理（最快）

### 使用透明PNG图片

1. **Shift + A** → Mesh → Plane
2. **S** 缩放到 **0.03**
3. Material Properties → **New**
4. 节点设置：
   ```
   Shift + A → Texture → Image Texture
   加载花朵PNG图片（带透明背景）
   
   连接：
   Image Texture (Color) → Principled BSDF (Base Color)
   Image Texture (Alpha) → Principled BSDF (Alpha)
   
   设置：
   Blend Mode: Alpha Clip
   ```

### 哪里找花朵纹理？
- **Polyhaven.com** - 搜索 "flower"
- **Textures.com** - 搜索 "orange blossom"
- 或者用AI生成：Midjourney/DALL-E生成白色五瓣花

## 不同生长阶段的花/果

### 阶段3：开花期（Flowering）
- 放置 **10-15朵白色小花**
- 分布在树枝末端
- 不放橘子

### 阶段4：结果期（Fruiting）
- 放置 **5-8个小绿球**（未成熟的橘子）
- 大小：0.05（比成熟橘子小）
- 颜色：浅绿色 RGB(0.6, 0.8, 0.3)
- 可以保留1-2朵花（还在开花）

### 阶段5：成熟期（Harvest）
- 放置 **5-8个橙色橘子**
- 大小：0.12（正常大小）
- 颜色：橙色 RGB(1.0, 0.5, 0.1)
- 不放花

## 快速制作流程

### 创建3个树模型变体

**1. 开花期树（Flowering）**
```
树干 + 树枝 + 树冠 + 10朵白花
```

**2. 结果期树（Fruiting）**
```
树干 + 树枝 + 树冠 + 5个小绿果 + 2朵白花
```

**3. 成熟期树（Harvest）**
```
树干 + 树枝 + 树冠 + 8个橙色橘子
```

### 导出3个FBX
```
SimpleOrangeTree_Flowering.fbx
SimpleOrangeTree_Fruiting.fbx
SimpleOrangeTree_Harvest.fbx
```

### Unity中切换
在你的生长系统中，根据阶段加载不同的模型。

## 实际操作建议

1. **先做成熟期树**（有橘子的）
2. **复制一份，删除橘子，加花朵**（开花期）
3. **再复制一份，加小绿果**（结果期）
4. **分别导出3个FBX**
5. **在Unity中根据生长阶段切换模型**

这样你就有完整的生长周期视觉效果了！🌸🍊
