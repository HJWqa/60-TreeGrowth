🍊 橘子树快速生成方法（无需菜单）
========================================

如果 TreePlanQAQ 菜单没有显示，使用这个更简单的方法！

========================================
⚡ 超级简单的方法（推荐）
========================================

第1步: 创建生成器
-----------------
1. 在Unity场景中，右键 → Create Empty
2. 命名为 "OrangeTreeGenerator"
3. 在Inspector中，点击 "Add Component"
4. 搜索并添加 "OrangeTreeQuickGenerator"

第2步: 生成所有橘子树
-----------------
1. 选中刚才创建的物体
2. 在Inspector中找到 OrangeTreeQuickGenerator 组件
3. 右键点击组件标题
4. 选择 "生成所有阶段"
5. ✅ 完成！所有10个阶段会在场景中显示

第3步: 保存为预制体
-----------------
1. 继续右键点击组件
2. 选择 "保存为预制体"
3. ✅ 所有预制体自动保存到:
   Assets/Prefabs/Plants/OrangeTrees/

========================================
🎯 详细步骤（带图文说明）
========================================

步骤1: 创建生成器物体
--------------------
Unity场景中:
   Hierarchy → 右键 → Create Empty
   或按快捷键: Ctrl+Shift+N (Windows)

步骤2: 添加生成器组件
--------------------
选中空物体 → Inspector → Add Component
输入: OrangeTreeQuickGenerator
点击添加

步骤3: 使用右键菜单
--------------------
在 OrangeTreeQuickGenerator 组件上右键，会看到:
   ✅ 生成橘子树 - 生成单个阶段
   ✅ 生成所有阶段 - 生成全部10个阶段
   ✅ 清除所有模型 - 删除已生成的模型
   ✅ 保存为预制体 - 保存到Project

步骤4: 生成模型
--------------------
右键 → "生成所有阶段"
等待几秒，场景中会出现10棵树，横向排列

步骤5: 保存预制体
--------------------
右键 → "保存为预制体"
Console会显示保存进度
完成后Project窗口会自动高亮文件夹

========================================
🎨 自定义设置
========================================

在 Inspector 中可以调整:

Stage To Generate:
   - 选择要生成的单个阶段
   - 默认: MatureTree（成树）

Generate All Stages:
   - 勾选后会生成所有阶段
   - 默认: false

Spacing:
   - 阶段之间的间距
   - 默认: 3单位
   - 建议: 2-5单位

========================================
📝 右键菜单功能说明
========================================

1. "生成橘子树"
   - 生成 Stage To Generate 选择的阶段
   - 会清除之前生成的模型
   - 模型出现在物体位置

2. "生成所有阶段"
   - 生成全部10个阶段
   - 横向排列，间距由 Spacing 控制
   - 会清除之前生成的模型

3. "清除所有模型"
   - 删除所有子对象
   - 清空场景中的树模型
   - 不影响已保存的预制体

4. "保存为预制体"
   - 将所有子对象保存为预制体
   - 自动创建输出文件夹
   - 文件名格式: OrangeTree_[阶段名].prefab
   - 保存位置: Assets/Prefabs/Plants/OrangeTrees/

========================================
💡 使用技巧
========================================

技巧1: 快速预览
--------------
1. 生成所有阶段
2. 在Scene视图中查看
3. 满意后再保存为预制体

技巧2: 单独调整
--------------
1. 选择 Stage To Generate
2. 点击"生成橘子树"
3. 查看单个阶段
4. 调整参数后重新生成

技巧3: 批量保存
--------------
1. 生成所有阶段
2. 在Hierarchy中手动调整位置
3. 一次性保存为预制体

技巧4: 重复使用
--------------
1. 保留这个生成器物体
2. 需要时随时生成
3. 可以创建多个生成器

========================================
🐛 常见问题
========================================

Q: 找不到 OrangeTreeQuickGenerator 组件？
A: 等待Unity编译完成，查看Console是否有错误

Q: 右键没有菜单？
A: 确保右键点击的是组件标题，不是字段

Q: 生成的模型看不见？
A: 检查Scene视图，可能需要按F键聚焦

Q: 保存预制体失败？
A: 检查输出路径权限，手动创建文件夹

Q: 模型是粉红色？
A: 检查项目渲染管线设置

========================================

4. 右键组件 → "保存为预制体"
5. 完成！

只需要4步，不需要任何菜单！
