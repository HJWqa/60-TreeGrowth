# 🍊 橘子树模型集成步骤指南

## 📋 完整集成流程

按照以下步骤，将橘子树模型完整集成到你的果树生长模拟系统中。

---

## 第一步：生成橘子树预制体 ⚡

### 方法A：快速生成（推荐）

1. 打开Unity编辑器
2. 顶部菜单栏选择：**TreePlanQAQ** → **Quick Generate All Orange Trees**
3. 在弹出的对话框中点击"生成"
4. 等待进度条完成（约5-10秒）
5. 完成后会自动打开输出文件夹

### 方法B：使用可视化窗口

1. 顶部菜单栏选择：**TreePlanQAQ** → **Generate Orange Tree Models**
2. 在弹出的窗口中点击"🌳 生成所有橘子树模型"
3. 等待生成完成

### 验证生成结果

在Project窗口中导航到：
```
Assets/Prefabs/Plants/OrangeTrees/
```

你应该看到10个预制体文件：
- ✅ OrangeTree_Seed.prefab
- ✅ OrangeTree_Sprout.prefab
- ✅ OrangeTree_Seedling.prefab
- ✅ OrangeTree_YoungTree.prefab
- ✅ OrangeTree_MatureTree.prefab
- ✅ OrangeTree_Flowering.prefab
- ✅ OrangeTree_Fruiting.prefab
- ✅ OrangeTree_Harvest.prefab
- ✅ OrangeTree_Dormant.prefab
- ✅ OrangeTree_Dead.prefab

---

## 第二步：配置PlantGrowthController 🎮

### 1. 打开场景

打开你的主场景（通常是 `OutdoorsScene.unity`）

### 2. 找到PlantGrowthController对象

在Hierarchy窗口中找到包含 `PlantGrowthController` 组件的GameObject

### 3. 配置Stage Configs

在Inspector窗口中：

1. 找到 **Stage Configs** 列表
2. 确保列表有10个元素（对应10个生长阶段）
3. 如果没有，点击"+"按钮添加

### 4. 拖拽预制体

按照以下对应关系，将预制体拖拽到对应的Stage Model字段：

| Stage Config 索引 | Stage | 预制体文件 |
|-------------------|-------|-----------|
| 0 | Seed | OrangeTree_Seed.prefab |
| 1 | Sprout | OrangeTree_Sprout.prefab |
| 2 | Seedling | OrangeTree_Seedling.prefab |
| 3 | YoungTree | OrangeTree_YoungTree.prefab |
| 4 | MatureTree | OrangeTree_MatureTree.prefab |
| 5 | Flowering | OrangeTree_Flowering.prefab |
| 6 | Fruiting | OrangeTree_Fruiting.prefab |
| 7 | Harvest | OrangeTree_Harvest.prefab |
| 8 | Dormant | OrangeTree_Dormant.prefab |
| 9 | Dead | OrangeTree_Dead.prefab |

### 5. 设置过渡时间（可选）

为每个阶段设置 `Transition Duration`（建议值）：
- Seed → Sprout: 1.0秒
- Sprout → Seedling: 1.5秒
- Seedling → YoungTree: 2.0秒
- YoungTree → MatureTree: 2.5秒
- MatureTree → Flowering: 2.0秒
- Flowering → Fruiting: 1.5秒
- Fruiting → Harvest: 1.0秒
- 其他: 1.0秒

---

## 第三步：配置生长参数 📊

### 1. 使用橘子树配置

在PlantGrowthController的Inspector中：

1. 找到 **Growth Config** 字段
2. 将 `OrangeTreeGrowthConfig.asset` 拖拽到这个字段
3. 或者点击圆圈图标，在弹出窗口中选择 `OrangeTreeGrowthConfig`

### 2. 调整参数（可选）

双击 `OrangeTreeGrowthConfig.asset` 打开配置：

**基础参数**：
- Plant Name: 橘子树
- Base Growth Rate: 1.2（基础生长速度）
- Logistic Growth Rate: 0.6（逻辑增长率）

**环境参数**：
- Temperature Optimal: 25°C（最适温度）
- Temperature Tolerance: 12°C（温度容忍度）
- Humidity Optimal: 65%（最适湿度）
- Humidity Tolerance: 25%（湿度容忍度）

**阶段阈值**：
- Sprout: 10
- Seedling: 25
- Young Tree: 50
- Mature Tree: 75
- Flowering: 90
- Fruiting: 95
- Harvest: 100

### 4. 验证所有阶段

可以通过以下方式快速测试所有阶段：

**方法A：手动调整生长值**
```csharp
// 在PlantGrowthController中添加测试代码
[ContextMenu("Test All Stages")]
void TestAllStages()
{
    StartCoroutine(TestStagesCoroutine());
}

IEnumerator TestStagesCoroutine()
{
    float[] thresholds = {0, 10, 25, 50, 75, 90, 95, 100};
    foreach (float threshold in thresholds)
    {
        SetGrowth(threshold);
        yield return new WaitForSeconds(3f);
    }
}
```

**方法B：使用Inspector**
在运行时，直接在Inspector中修改 `Current Growth` 的值

## 第五步：优化和调整 🔧

### 性能优化

如果场景中有多棵树：

1. **使用对象池**
   - 复用模型而不是频繁创建/销毁

## 常见问题解决 🐛

### 问题1：预制体是粉红色的

**原因**：材质shader未找到

**解决方案**：
1. 检查项目的渲染管线（Edit → Project Settings → Graphics）
2. 确保使用HDRP、URP或Built-in
3. 重新生成预制体

### 问题2：模型太小或太大

**原因**：缩放不匹配

**解决方案**：
1. 选中PlantGrowthController对象
2. 调整Transform的Scale值
3. 或者修改OrangeTreeGenerator.cs中的尺寸参数

### 问题3：阶段切换不平滑

**原因**：过渡时间太短

**解决方案**：
1. 增加Stage Configs中的Transition Duration
2. 建议设置为1.5-2.5秒

### 问题4：生长速度太快/太慢

**原因**：生长参数不合适

**解决方案**：
1. 调整OrangeTreeGrowthConfig中的Base Growth Rate
2. 调整Environment Simulator的Update Interval

### 问题5：某个阶段不显示

**原因**：预制体未正确配置

**解决方案**：
1. 检查Stage Configs中对应阶段的Stage Model是否已设置
2. 确保预制体文件存在且未损坏
3. 重新生成该阶段的预制体

## 高级功能 🚀

### 1. 添加多棵橘子树

```csharp
// 在场景中创建多棵树
for (int i = 0; i < 5; i++)
{
    GameObject tree = Instantiate(plantControllerPrefab);
    tree.transform.position = new Vector3(i * 5, 0, 0);

```

### 2. 实现收获功能

```csharp
public void HarvestOranges()
{
    if (currentStage == PlantStage.Harvest)
    {
        int orangeCount = Random.Range(20, 35);
        Debug.Log($"收获了 {orangeCount} 个橘子！");
        
        // 重置到种子阶段
        ResetPlant();
    }
}
```

### 3. 添加天气影响

```csharp
public void OnRainStart()
{
    // 下雨时增加湿度
    EnvironmentSimulator.Instance.SetHumidity(85f);
}

public void OnSunnyDay()
{
    // 晴天时增加光照
    EnvironmentSimulator.Instance.SetSunlight(900f);
}
```
