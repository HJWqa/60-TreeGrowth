1️⃣ 种子 (Seed)

2️⃣ 发芽 (Sprout)
   改进前: 茎 + 2片叶
   改进后: 保持不变
   变化: 无
   ✅ 已经足够清晰

3️⃣ 幼苗 (Seedling)
   改进前: 主茎 + 2层叶片
   改进后: 主茎 + 2层叶片 + 顶部嫩芽
   变化: +1个嫩芽
   ✨ 新增: 浅绿色嫩芽显示生长活力

4️⃣ 小树 (YoungTree)
   改进前: 树干 + 3根分支 + 1层树冠
   改进后: 树干 + 4根分支(带叶) + 2层树冠
   变化: +1根分支, +1层树冠, 分支末端带叶
   ✨ 新增: 分支末端叶簇，树形更饱满

5️⃣ 成树 (MatureTree)
   改进前: 树干 + 6根分支 + 4层树冠
   改进后: 树干+底部 + 8根分支 + 5层树冠 + 6个小分支
   变化: +底部, +2根主分支, +1层树冠, +6个小分支
   ✨ 新增: 
      - 树干底部加粗（根部效果）
      - 小分支细节（带叶簇）
      - 更茂密的树冠

6️⃣ 开花 (Flowering)
   改进前: 25朵花（花心+5瓣）
   改进后: 25朵花（花心+5瓣+5萼+花柄）
   变化: 每朵花增加6个对象
   ✨ 新增:
      - 绿色花萼（5片）
      - 花柄（连接茎）
      - 花瓣尺寸优化

7️⃣ 结果 (Fruiting)
   改进前: 20个青橘子（球形+蒂）
   改进后: 20个青橘子（扁球+蒂+底部）
   变化: 每个橘子增加1个底部细节
   ✨ 新增:
      - 扁球形状（更真实）
      - 底部花萼痕迹

8️⃣ 成熟 (Harvest)
   改进前: 30个橙橘子（球形+蒂）
   改进后: 30个橙橘子（扁球+蒂+底部+光泽）
   变化: 每个橘子增加1个底部细节+材质光泽
   ✨ 新增:
      - 扁球形状
      - 底部花萼痕迹
      - 光泽材质效果

9️⃣ 休眠 (Dormant)
   改进前: 树干 + 3根光秃分支
   改进后: 树干+底部 + 4根主分支(带分叉) + 8个细枝
   变化: +底部, +1根主分支, 每根分支+2-3分叉, +8细枝
   ✨ 新增:
      - 树干底部
      - 分支上的小分叉
      - 细小枯枝
      - 更深的树皮颜色

🔟 枯死 (Dead)
   改进前: 倾斜树干 + 1根断枝
   改进后: 倾斜树干+底部 + 2根断枝 + 5个残枝 + 3条裂纹
   变化: +底部, +1根断枝, +5残枝, +3裂纹
   ✨ 新增:
      - 腐朽根部
      - 第2根断枝
      - 残留枯枝
      - 树干裂纹效果
      - 倾斜角度增加
