⚡ 快速命令
========================================

生成所有橘子树:
   Unity菜单 → TreePlanQAQ → Quick Generate All Orange Trees

打开生成器窗口:
   Unity菜单 → TreePlanQAQ → Generate Orange Tree Models

打开橘子树文件夹:
   Unity菜单 → TreePlanQAQ → Open Orange Trees Folder
