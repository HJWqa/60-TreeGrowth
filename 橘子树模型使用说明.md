# 🍊 橘子树模型生成器使用说明

## 功能介绍

这是一个专门为橘子树设计的程序化模型生成系统，可以生成橘子树从种子到成熟的完整生长过程。

## 橘子树特征

生成的橘子树模型包含以下真实特征：

- **深绿色叶子** - 橘子树特有的深绿色叶片
- **白色花朵** - 开花阶段会出现白色小花
- **果实变化** - 从青色小果实逐渐变为橙色成熟橘子
- **分支结构** - 真实的树干和分支系统
- **生长阶段** - 10个完整的生长阶段

## 生长阶段说明

| 阶段 | 名称 | 特征 |
|------|------|------|
| 1 | 种子 (Seed) | 椭圆形的橘子种子 |
| 2 | 发芽 (Sprout) | 细嫩的茎和两片初生叶 |
| 3 | 幼苗 (Seedling) | 主茎和多层叶片 |
| 4 | 小树 (Young Tree) | 开始形成树干和分支 |
| 5 | 成树 (Mature Tree) | 茂密的树冠和粗壮树干 |
| 6 | 开花 (Flowering) | 树冠上开满白色小花 |
| 7 | 结果 (Fruiting) | 青色的小橘子开始生长 |
| 8 | 成熟 (Harvest) | 橙色和黄色的成熟橘子 |
| 9 | 休眠 (Dormant) | 落叶后只剩树干和枝条 |
| 10 | 枯死 (Dead) | 枯萎倾斜的树干 |

## 使用方法

### 方法一：使用编辑器窗口（推荐）

1. 在Unity编辑器顶部菜单栏选择：`TreePlanQAQ` → `Generate Orange Tree Models`
2. 在弹出的窗口中：
   - 可以点击 **"生成所有橘子树模型"** 一键生成全部10个阶段
   - 或者点击单独的阶段按钮，只生成特定阶段的模型
3. 生成的预制体会自动保存到 `Assets/Prefabs/Plants/OrangeTrees/` 目录

### 方法二：通过代码调用

```csharp
using TreePlanQAQ.Plant;

// 生成特定阶段的橘子树模型
GameObject orangeTree = OrangeTreeGenerator.CreateOrangeTreeModel(PlantStage.Harvest);

// 设置位置
orangeTree.transform.position = new Vector3(0, 0, 0);
```

## 输出文件

生成的预制体文件命名格式：`OrangeTree_{阶段名}.prefab`

例如：
- `OrangeTree_Seed.prefab`
- `OrangeTree_Flowering.prefab`
- `OrangeTree_Harvest.prefab`

## 与现有系统集成

生成的橘子树模型可以直接用于现有的植物生长系统：

1. 打开场景中的 `PlantGrowthController` 对象
2. 在 Inspector 面板的 `Stage Configs` 列表中
3. 将生成的橘子树预制体拖拽到对应阶段的 `Stage Model` 字段
4. 运行游戏，橘子树会根据温湿度自动生长

## 自定义修改

如果需要调整橘子树的外观，可以修改 `OrangeTreeGenerator.cs` 中的颜色常量：

```csharp
// 在文件顶部找到这些颜色定义
private static readonly Color OrangeLeafColor = new Color(0.15f, 0.5f, 0.15f);
private static readonly Color OrangeFruitOrange = new Color(1f, 0.5f, 0.1f);
// ... 等等
```

## 技术细节

- **程序化生成** - 所有模型都是运行时通过代码创建的基础几何体组合
- **材质系统** - 自动适配 HDRP/URP/Standard 渲染管线
- **性能优化** - 使用简单几何体，性能开销小
- **可扩展性** - 易于修改和扩展新的生长阶段

## 注意事项

1. 生成的模型是占位模型，适合原型开发和测试
2. 如需更精细的模型，建议使用专业3D建模软件制作
3. 确保输出路径存在，工具会自动创建不存在的文件夹
4. 生成过程会覆盖同名的预制体文件

## 故障排除

**问题：菜单中找不到生成工具**
- 确保 `OrangeTreeGeneratorEditor.cs` 文件在 `Editor` 文件夹中
- 等待Unity完成脚本编译

**问题：生成的模型是粉红色的**
- 说明材质shader未找到
- 检查项目的渲染管线设置（HDRP/URP/Built-in）

**问题：预制体保存失败**
- 检查输出路径是否有写入权限
- 确保路径以 `Assets/` 开头
