# 新橘子树生长系统设计

## 系统概述
一个简洁、易用的橘子树生长模拟系统，支持8个生长阶段，使用手工制作的3D模型。

## 核心功能

### 1. 生长阶段管理
- 8个生长阶段：种子 → 发芽 → 幼苗 → 小树 → 成树 → 开花 → 结果 → 成熟
- 平滑的阶段过渡动画
- 可配置的生长速度

### 2. 环境系统
- 温度影响生长速度
- 湿度影响生长速度
- 光照影响生长速度
- 实时环境监测面板

### 3. 视觉效果
- 阶段切换时的过渡动画（缩放、淡入淡出）
- 可选的粒子特效
- 平滑的模型切换

## 系统架构

### 核心组件

1. **OrangeTreeController** - 主控制器
   - 管理生长进度
   - 控制阶段切换
   - 处理模型显示

2. **EnvironmentManager** - 环境管理器
   - 管理环境参数（温度、湿度、光照）
   - 计算生长速率
   - 提供环境数据

3. **GrowthConfig** - 生长配置（ScriptableObject）
   - 定义8个阶段的参数
   - 配置生长速率
   - 设置环境影响因子

4. **UIController** - UI控制器
   - 显示当前阶段
   - 显示生长进度
   - 显示环境参数

## 使用流程

1. **导入模型** - 将8个阶段的FBX模型导入Unity
2. **创建预制体** - 为每个阶段创建预制体
3. **配置系统** - 在Config中配置阶段参数
4. **添加到场景** - 将OrangeTreeController添加到场景
5. **运行测试** - 点击Play测试生长过程
